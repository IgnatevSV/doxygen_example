Проект создан при помощи Doxywizard.

Doxywizard генерирует конфиг.файл, по которому затем создается проект.
В Doxywizard есть несколько режимов создания конфиг.файла - Wizard и Expert. В режиме Wizard можно задать основные настройки конфиг.файла, в Expert - расширенные настройки конфиг.файла.
Обязательно необходимо указать каталог источника и каталог для сгенерированных файлов.
Для генерации сайта необходимо перейти на вкладку Run и нажать на кнопку Run Doxygen.
После того, как Doxygen закончит генерацию файлов, можно просмотреть сайт нажав кнопку Show HTML Output или же запустив проект из каталога сгенерированных файлов.

Исходные файлы в данном проекте - source_for_doxygen, где:
Task_Dialogues - исходные файлы самого проекта.
dialogues_system_description - файлы описания проекта в формате Markdown.
